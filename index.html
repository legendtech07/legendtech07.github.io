HTML Tags Reference

<!DOCTYPE html>

Defines the document type and HTML version being used. This declaration must be the very first thing in your HTML document.

<!DOCTYPE html>
Example:
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My Heading</h1>
</body>
</html>

<html>

The root element of an HTML page that contains all other HTML elements.

<html>...</html>
Example:
<html lang="en">
<!-- All content goes here -->
</html>

<head>

Contains meta-information about the document, including its title, links to scripts and style sheets.

<head>...</head>
Example:
<head>
    <title>My Page</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="styles.css">
</head>

<title>

Defines the title of the document, shown in the browser's title bar or tab.

<title>Page Title</title>
Example:
<title>My Awesome Website</title>

<body>

Contains the visible page content that users see in their browser.

<body>...</body>
Example:
<body>
    <h1>Welcome to my site</h1>
    <p>This is my content.</p>
</body>

<h1> to <h6>

HTML headings from most important (h1) to least important (h6).

<h1>Heading 1</h1>
<h2>Heading 2</h2>
...
<h6>Heading 6</h6>
Example:
<h1>Main Title</h1>
<h2>Section Heading</h2>
<h3>Subsection</h3>

<p>

Defines a paragraph of text.

<p>Paragraph text</p>
Example:
<p>This is a paragraph of text that will appear on my webpage.</p>
<p>Here is another paragraph.</p>

<a>

Defines a hyperlink to link from one page to another.

<a href="url">link text</a>
Example:
<a href="https://www.example.com">Visit Example.com</a>
<a href="about.html">About Us</a>
<a href="#section2">Jump to Section 2</a>

<img>

Embeds an image in the document. This is a self-closing tag.

<img src="image.jpg" alt="description">
Example:
<img src="logo.png" alt="Company Logo" width="200" height="100">
<img src="photos/sunset.jpg" alt="Sunset over mountains">

<div>

Defines a division or section in an HTML document, used as a container for other elements.

<div>...</div>
Example:
<div class="header">
    <h1>Welcome</h1>
</div>

<div id="content">
    <p>Main content goes here.</p>
</div>

<span>

An inline container used to mark up a part of text or document.

<span>...</span>
Example:
<p>This is <span class="highlight">important</span> text.</p>

<ul>, <ol>, <li>

Defines lists: unordered (bulleted) with <ul>, ordered (numbered) with <ol>, and list items with <li>.

<ul>
  <li>Item</li>
</ul>

<ol>
  <li>First item</li>
</ol>
Example:
<ul>
    <li>Coffee</li>
    <li>Tea</li>
</ul>

<ol>
    <li>First step</li>
    <li>Second step</li>
</ol>

<table>, <tr>, <th>, <td>

Defines a table with rows and cells.

  • <table>: Defines the table
  • <tr>: Defines a table row
  • <th>: Defines a header cell
  • <td>: Defines a standard data cell
<table>
  <tr>
    <th>Header</th>
  </tr>
  <tr>
    <td>Data</td>
  </tr>
</table>
Example:
<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

<form>

Defines an HTML form for user input.

<form action="url" method="post/get">...</form>
Example:
<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

<input>

Defines an input control for forms. The type attribute specifies the kind of input.

<input type="text|password|checkbox|radio|submit|etc." name="field-name">
Example:
<input type="text" name="username" placeholder="Enter username">
<input type="password" name="pwd">
<input type="checkbox" name="subscribe" checked>
<input type="radio" name="gender" value="male">
<input type="submit" value="Send">

<button>

Defines a clickable button.

<button type="button|submit|reset">Label</button>
Example:
<button type="button" onclick="alert('Clicked!')">Click Me</button>
<button type="submit">Submit Form</button>

<textarea>

Defines a multi-line text input control.

<textarea rows="4" cols="50">Default text</textarea>
Example:
<textarea name="message" rows="10" cols="30">
The cat was playing in the garden.
</textarea>

<select>, <option>

Defines a drop-down list of options.

<select name="name">
  <option value="value1">Option 1</option>
  <option value="value2">Option 2</option>
</select>
Example:
<select name="cars">
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
    <option value="mercedes">Mercedes</option>
</select>

<label>

Defines a label for an <input> element.

<label for="input-id">Label text</label>
Example:
<label for="username">Username:</label>
<input type="text" id="username" name="username">

<header>, <footer>, <nav>, <section>, <article>, <aside>

HTML5 semantic elements that define parts of a document:

  • <header>: Introductory content
  • <footer>: Footer for a document or section
  • <nav>: Navigation links
  • <section>: Thematic grouping of content
  • <article>: Self-contained composition
  • <aside>: Content aside from the main content
<header>...</header>
<footer>...</footer>
<nav>...</nav>
<section>...</section>
<article>...</article>
<aside>...</aside>
Example:
<header>
    <h1>Website Title</h1>
    <nav>
        <a href="/">Home</a>
        <a href="/about">About</a>
    </nav>
</header>

<section>
    <article>
        <h2>Blog Post Title</h2>
        <p>Content...</p>
    </article>
</section>

<footer>
    <p>Copyright 2023</p>
</footer>

CSS Properties Reference

color

Sets the color of text.

color: value;
Example:
body {
    color: #333333; /* Dark gray */
}

h1 {
    color: red;
}

.special {
    color: rgb(0, 150, 200);
}

background-color

Sets the background color of an element.

background-color: value;
Example:
body {
    background-color: #f5f5f5;
}

header {
    background-color: rgba(0, 0, 0, 0.8);
}

.button {
    background-color: hsl(120, 100%, 50%);
}

font-family

Specifies the font for an element.

font-family: font1, font2, generic-family;
Example:
body {
    font-family: Arial, Helvetica, sans-serif;
}

h1 {
    font-family: "Times New Roman", Times, serif;
}

.code {
    font-family: "Courier New", Courier, monospace;
}

font-size

Sets the size of the font.

font-size: value;
Example:
body {
    font-size: 16px;
}

h1 {
    font-size: 2em; /* 2 times parent element's font size */
}

.small {
    font-size: 0.8rem; /* Relative to root element */
}

font-weight

Sets how thick or thin characters in text should be displayed.

font-weight: normal|bold|bolder|lighter|number;
Example:
p {
    font-weight: normal;
}

strong {
    font-weight: bold;
}

.heading {
    font-weight: 700; /* Same as bold */
}

text-align

Specifies the horizontal alignment of text.

text-align: left|right|center|justify;
Example:
h1 {
    text-align: center;
}

article {
    text-align: justify;
}

.right {
    text-align: right;
}

margin

Sets the margin area on all four sides of an element.

margin: all;
margin: top-bottom left-right;
margin: top right bottom left;
Example:
body {
    margin: 0; /* No margin */
}

div {
    margin: 20px; /* All sides */
}

p {
    margin: 10px 20px; /* Top/bottom, left/right */
}

.special {
    margin: 5px 10px 15px 20px; /* Top, right, bottom, left */
}

margin

Sets the margin area on all four sides of an element.

margin: all;
margin: top-bottom left-right;
margin: top right bottom left;
Example:
body {
    margin: 0; /* No margin */
}

div {
    margin: 20px; /* All sides */
}

p {
    margin: 10px 20px; /* Top/bottom, left/right */
}

.special {
    margin: 5px 10px 15px 20px; /* Top, right, bottom, left */
}

padding

Sets the padding area on all four sides of an element.

padding: all;
padding: top-bottom left-right;
padding: top right bottom left;
Example:
.box {
    padding: 10px;
}

.card {
    padding: 20px 15px;
}

.button {
    padding: 5px 10px 15px 20px;
}

border

Sets the border of an element (width, style, and color).

border: width style color;
Example:
div {
    border: 1px solid black;
}

img {
    border: 2px dotted #ff0000;
}

.special {
    border: 5px double rgba(0, 0, 255, 0.5);
}

width & height

Sets the width and height of an element.

width: value;
height: value;
Example:
.container {
    width: 80%;
    height: 500px;
}

img {
    width: 300px;
    height: auto; /* Maintain aspect ratio */
}

.box {
    width: min(80%, 1000px);
    height: max(50vh, 300px);
}

display

Specifies the display behavior of an element.

Common values:

  • block: Takes full width, starts on new line
  • inline: Flows with text, no width/height
  • inline-block: Flows with text but accepts width/height
  • flex: Flexbox layout
  • grid: CSS Grid layout
  • none: Completely removed
display: value;
Example:
div {
    display: block;
}

span {
    display: inline;
}

.menu {
    display: flex;
}

.hidden {
    display: none;
}

position

Specifies the type of positioning method used for an element.

Values:

  • static: Default, follows normal flow
  • relative: Positioned relative to normal position
  • absolute: Positioned relative to nearest positioned ancestor
  • fixed: Positioned relative to viewport
  • sticky: Hybrid of relative and fixed
position: value;
Example:
.box {
    position: relative;
    top: 20px;
    left: 30px;
}

.modal {
    position: fixed;
    top: 0;
    right: 0;
}

.header {
    position: sticky;
    top: 0;
}

float

Specifies whether an element should float to the left, right, or not at all.

float: left|right|none;
Example:
img {
    float: left;
    margin-right: 15px;
}

.sidebar {
    float: right;
    width: 30%;
}

flexbox properties

Flexbox layout properties for creating flexible layouts.

Key properties:

  • display: flex: Enables flexbox
  • flex-direction: Row or column
  • justify-content: Main axis alignment
  • align-items: Cross axis alignment
  • flex-wrap: Single or multiple lines
  • flex: Shorthand for flex-grow, flex-shrink, flex-basis
display: flex;
flex-direction: row|column;
justify-content: flex-start|flex-end|center|space-between|space-around;
align-items: stretch|flex-start|flex-end|center|baseline;
flex: grow shrink basis;
Example:
.container {
    display: flex;
    flex-direction: row;
    justify-content: space-between;
    align-items: center;
}

.item {
    flex: 1 0 200px;
}

.column {
    flex-direction: column;
}

grid properties

CSS Grid properties for two-dimensional layouts.

Key properties:

  • display: grid: Enables grid
  • grid-template-columns: Defines columns
  • grid-template-rows: Defines rows
  • gap: Space between items
  • grid-column/grid-row: Item placement
  • grid-area: Named grid areas
display: grid;
grid-template-columns: sizes;
grid-template-rows: sizes;
gap: size;
grid-column: start/end;
grid-row: start/end;
Example:
.container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-rows: 100px auto;
    gap: 20px;
}

.header {
    grid-column: 1 / 4;
}

.sidebar {
    grid-row: 2 / 3;
    grid-column: 1 / 2;
}

transition

Creates smooth transitions between property changes.

transition: property duration timing-function delay;
Example:
.button {
    background-color: blue;
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: darkblue;
}

.box {
    transition: all 0.5s cubic-bezier(0.4, 0, 0.2, 1);
}

transform

Applies 2D or 3D transformations to an element.

Common functions:

  • translate(): Move element
  • rotate(): Rotate element
  • scale(): Change size
  • skew(): Skew element
transform: function(value);
Example:
.element {
    transform: translateX(50px);
}

.rotate {
    transform: rotate(45deg);
}

.grow {
    transform: scale(1.2);
}

.combined {
    transform: translate(10px, 20px) rotate(10deg);
}

animation

Creates animations by gradually changing styles.

Requires @keyframes rule to define the animation sequence.

@keyframes name {
  from { properties }
  to { properties }
}

animation: name duration timing-function delay iteration-count direction fill-mode;
Example:
@keyframes slidein {
    from { transform: translateX(-100%); }
    to { transform: translateX(0); }
}

.box {
    animation: slidein 1s ease-in-out;
}

@keyframes pulse {
    0% { transform: scale(1); }
    50% { transform: scale(1.1); }
    100% { transform: scale(1); }
}

.button {
    animation: pulse 2s infinite;
}

media queries

Applies styles based on device characteristics (like screen size).

@media media-type and (condition) {
  /* CSS rules */
}
Example:
/* Default styles */
.container {
    width: 100%;
}

/* For screens wider than 768px */
@media (min-width: 768px) {
    .container {
        width: 750px;
    }
}

/* For screens wider than 992px */
@media (min-width: 992px) {
    .container {
        width: 970px;
    }
}

/* Print styles */
@media print {
    .no-print {
        display: none;
    }
}

JavaScript Reference

Variables

Variables are containers for storing data values.

JavaScript has three ways to declare variables:

  • var: Old way, function-scoped
  • let: Block-scoped, can be reassigned
  • const: Block-scoped, cannot be reassigned
var name = value;
let name = value;
const name = value;
Example:
var oldVariable = "I'm old school";
let changeable = "I can change";
const constantValue = "I stay the same";

// Reassignment
changeable = "New value"; // OK
// constantValue = "New value"; // Error!

// Scope example
if (true) {
    let insideBlock = "I'm here";
    var insideToo = "Me too";
}
// console.log(insideBlock); // Error
console.log(insideToo); // Works

Data Types

JavaScript has dynamic types. The same variable can hold different data types.

Primitive types:

  • Number: Integer or floating point
  • String: Text
  • Boolean: true/false
  • null: Intentional empty value
  • undefined: Variable not assigned
  • Symbol: Unique identifier
  • BigInt: Large integers

Object types: Object, Array, Date, etc.

// Primitives
let num = 42;
let str = "Hello";
let bool = true;

// Objects
let obj = { key: "value" };
let arr = [1, 2, 3];
Example:
// Number
let age = 25;
let price = 9.99;

// String
let name = "Alice";
let greeting = 'Hello';

// Boolean
let isActive = true;
let isComplete = false;

// Null
let emptyValue = null;

// Undefined
let notAssigned;

// Object
let person = {
    firstName: "John",
    lastName: "Doe"
};
                        
// Array
let colors = ["red", "green", "blue"];

Operators

Operators perform operations on variables and values.

Types of operators:

  • Arithmetic: +, -, *, /, %, **, ++, --
  • Assignment: =, +=, -=, *=, /=
  • Comparison: ==, ===, !=, !==, >, <, >=, <=
  • Logical: &&, ||, !
  • Ternary: condition ? expr1 : expr2
// Arithmetic
let sum = a + b;

// Assignment
x += 5; // x = x + 5

// Comparison
if (a === b) { ... }

// Logical
if (a && b) { ... }

// Ternary
let result = condition ? "yes" : "no";
Example:
// Arithmetic
let x = 10 + 5; // 15
let y = 10 % 3; // 1 (remainder)
let z = 2 ** 3; // 8 (exponentiation)

// Assignment
let a = 10;
a += 5; // a is now 15

// Comparison
console.log(5 == "5"); // true (loose equality)
console.log(5 === "5"); // false (strict equality)

// Logical
let isAdult = true;
let hasLicense = false;
console.log(isAdult && hasLicense); // false
console.log(isAdult || hasLicense); // true

// Ternary
let age = 20;
let canVote = age >= 18 ? "Yes" : "No"; // "Yes"

Conditional Statements

Conditional statements control behavior based on different conditions.

Types:

  • if: Executes code if condition is true
  • else: Executes code if if condition is false
  • else if: Specifies new condition if first is false
  • switch: Selects one of many code blocks to execute
// if-else
if (condition) {
  // code
} else if (condition2) {
  // code
} else {
  // code
}

// switch
switch(expression) {
  case x:
    // code
    break;
  case y:
    // code
    break;
  default:
    // code
}
Example:
// if-else
let hour = new Date().getHours();
let greeting;

if (hour < 12) {
    greeting = "Good morning";
} else if (hour < 18) {
    greeting = "Good afternoon";
} else {
    greeting = "Good evening";
}

// switch
let day;
switch (new Date().getDay()) {
    case 0:
        day = "Sunday";
        break;
    case 1:
        day = "Monday";
        break;
    // ... other cases
    default:
        day = "Unknown";
}

Loops

Loops execute a block of code repeatedly while a condition is true.

Types:

  • for: Loops through code a number of times
  • while: Loops while condition is true
  • do...while: Loops once, then repeats while condition is true
  • for...in: Loops through object properties
  • for...of: Loops through iterable values (arrays, strings)
// for loop
for (init; condition; increment) {
  // code
}

// while loop
while (condition) {
  // code
}

// for...of
for (item of iterable) {
  // code
}
Example:
// for loop
for (let i = 0; i < 5; i++) {
    console.log(i); // 0, 1, 2, 3, 4
}

// while loop
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

// for...of with array
let colors = ["red", "green", "blue"];
for (let color of colors) {
    console.log(color);
}

// for...in with object
let person = {name: "John", age: 30};
for (let key in person) {
    console.log(key + ": " + person[key]);
}

Functions

Functions are blocks of code designed to perform particular tasks.

Types:

  • Function declarations: Defined with function keyword
  • Function expressions: Assigned to variables
  • Arrow functions: Shorter syntax, no own this
// Function declaration
function name(parameters) {
  // code
  return value;
}

// Function expression
const name = function(parameters) {
  // code
};

// Arrow function
const name = (parameters) => {
  // code
};
Example:
// Function declaration
function greet(name) {
    return "Hello, " + name;
}
console.log(greet("Alice")); // "Hello, Alice"

// Function expression
const square = function(x) {
    return x * x;
};
console.log(square(5)); // 25

// Arrow function
const add = (a, b) => a + b;
console.log(add(2, 3)); // 5

// Arrow function with multiple statements
const greetUser = (name) => {
    const message = "Hello, " + name;
    return message;
};

// Default parameters
function multiply(a, b = 1) {
    return a * b;
}
console.log(multiply(5)); // 5
console.log(multiply(5, 2)); // 10

Arrays

Arrays are used to store multiple values in a single variable.

Common methods:

  • push()/pop(): Add/remove from end
  • shift()/unshift(): Add/remove from start
  • slice(): Extract portion
  • splice(): Add/remove at position
  • map(): Transform elements
  • filter(): Select elements
  • reduce(): Reduce to single value
let arr = [value1, value2, ...];
arr.method();
Example:
// Creating arrays
let fruits = ["Apple", "Banana"];
let numbers = [1, 2, 3, 4, 5];

// Accessing elements
console.log(fruits[0]); // "Apple"

// Modifying arrays
fruits.push("Orange"); // Add to end
fruits.pop(); // Remove from end
fruits.unshift("Strawberry"); // Add to start
fruits.shift(); // Remove from start

// Iterating
fruits.forEach(function(item, index) {
    console.log(index, item);
});

// Mapping
let doubled = numbers.map(num => num * 2);

// Filtering
let evens = numbers.filter(num => num % 2 === 0);

// Reducing
let sum = numbers.reduce((total, num) => total + num, 0);

// Spread operator
let moreFruits = [...fruits, "Mango"];

Objects

Objects are collections of key-value pairs.

Common operations:

  • Property access: dot notation or brackets
  • Object methods: Functions as property values
  • Object destructuring: Unpack properties
  • Spread operator: Copy/merge objects
let obj = {
  key: value,
  method() { ... }
};

obj.property;
obj["property"];
obj.method();
Example:
// Creating objects
let person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    fullName() {
        return this.firstName + " " + this.lastName;
    }
};

// Accessing properties
console.log(person.firstName); // "John"
console.log(person["lastName"]); // "Doe"
console.log(person.fullName()); // "John Doe"

// Adding properties
person.email = "john@example.com";

// Object destructuring
let { firstName, age } = person;
console.log(firstName, age); // "John" 30

// Spread operator
let updatedPerson = { ...person, age: 31 };

// Object.keys(), Object.values()
let keys = Object.keys(person);
let values = Object.values(person);

DOM Manipulation

The Document Object Model (DOM) represents the page as objects that JavaScript can manipulate.

Common methods:

  • Selecting elements: querySelector, getElementById
  • Modifying content: innerHTML, textContent
  • Modifying attributes: getAttribute, setAttribute
  • Modifying styles: style.property
  • Event listeners: addEventListener
  • Creating elements: createElement, appendChild
// Selecting
document.querySelector('css-selector');
document.getElementById('id');

// Modifying
element.textContent = 'text';
element.style.color = 'red';

// Events
element.addEventListener('click', function);
Example:
// Selecting elements
let header = document.querySelector('h1');
let button = document.getElementById('myButton');

// Modifying content
header.textContent = "New Title";

// Modifying styles
header.style.color = "blue";
header.style.fontSize = "24px";

// Adding event listener
button.addEventListener('click', function() {
    alert('Button clicked!');
});

// Creating and adding elements
let newDiv = document.createElement('div');
newDiv.textContent = "I'm new here!";
document.body.appendChild(newDiv);

// Form handling
let form = document.querySelector('form');
form.addEventListener('submit', function(event) {

    let input = document.getElementById('nameInput');

Events

Events are actions that happen in the browser that JavaScript can respond to.

Common events:

  • Mouse events: click, dblclick, mouseover, mouseout
  • Keyboard events: keydown, keyup, keypress
  • Form events: submit, change, focus, blur
  • Window events: load, resize, scroll
element.addEventListener(event, function);
element.onclick = function;
Example:
// Click event
document.getElementById('myButton').addEventListener('click', function() {
    console.log('Button clicked');
});

// Mouse events
let box = document.querySelector('.box');
box.addEventListener('mouseover', function() {
    this.style.backgroundColor = 'yellow';
});
box.addEventListener('mouseout', function() {
    this.style.backgroundColor = '';
});

// Keyboard event
document.addEventListener('keydown', function(event) {
    console.log('Key pressed:', event.key);
});

// Form submission
document.querySelector('form').addEventListener('submit', function(event) {
    event.preventDefault();
    console.log('Form submitted');
});

// Window events
window.addEventListener('resize', function() {
    console.log('Window resized to:', window.innerWidth, 'x', window.innerHeight);
});

Error Handling

Try...catch statements handle errors gracefully without stopping script execution.

try {
  // Code that might throw an error
} catch (error) {
  // Handle the error
} finally {
  // Execute always
}
Example:
// Basic error handling
try {
    // Code that might fail
    let result = riskyOperation();
    console.log(result);
} catch (error) {
    console.error("Something went wrong:", error.message);
} finally {
    console.log("This always runs");
}

// Throwing custom errors
function divide(a, b) {
    if (b === 0) {
        throw new Error("Division by zero");
    }
    return a / b;
}

try {
    console.log(divide(10, 0));
} catch (error) {
    console.error(error.message); // "Division by zero"
}
