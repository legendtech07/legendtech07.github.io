<!DOCTYPE html> HTML

Defines the document type and HTML version being used.

<!DOCTYPE html>
Example:
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
</body>
</html>
<a> HTML

Defines a hyperlink to link from one page to another.

<a href="url">link text</a>
Example:
<a href="https://example.com">Visit Example</a>
<a href="#section2">Jump to Section 2</a>

CSS Reference

â–¼
display CSS

Specifies the display behavior of an element.

display: block | inline | inline-block | flex | grid | none;
Example:
.block-element {
    display: block;
}

.flex-container {
    display: flex;
    justify-content: center;
}
padding CSS

Sets the padding area on all four sides of an element.

padding: all;
padding: top-bottom left-right;
padding: top right bottom left;
Example:
.box {
    padding: 20px; /* All sides */
}

.card {
    padding: 10px 20px; /* Top/bottom, left/right */
}

.special {
    padding: 5px 10px 15px 20px; /* Top, right, bottom, left */
}

Variables JavaScript

Containers for storing data values.

let variableName = value;
const constantName = value;
var oldVariable = value;
Example:
let age = 25;
const PI = 3.14159;
var name = "John";

age = 26; // Valid
// PI = 3.14; // Error (const can't be reassigned)
Functions JavaScript

Blocks of code designed to perform particular tasks.

function name(parameters) {
  // code
}

const name = function(parameters) {
  // code
};

const name = (parameters) => {
  // code
};
Example:
function greet(name) {
    return "Hello, " + name;
}

const square = function(x) {
    return x * x;
};

const add = (a, b) => a + b;
